# Installation

## Requirements

- Python >= 3.10
- `packaging` (only runtime dependency)

## Via pipx (recommended for CLI use)

```bash
pipx install do_i_need_to_upgrade
```

## Via uv tool

```bash
uv tool install do_i_need_to_upgrade
```

## Via pip / uv add (for library use)

```bash
pip install do_i_need_to_upgrade
# or
uv add do_i_need_to_upgrade
```
